# config.json — Configuração Local de Modelos

**Última atualização:** 2026-06-12

## Responsabilidade

Arquivo de configuração local usado como referência para modelos externos.

## Alteração de segurança

As chaves reais de API foram removidas e substituídas por placeholders:

- `COLOQUE_SUA_CHAVE_GROQ_AQUI`
- `COLOQUE_SUA_CHAVE_DEEPSEEK_AQUI`

Nunca salve chaves reais nesse arquivo antes de commitar ou enviar ao GitHub.
